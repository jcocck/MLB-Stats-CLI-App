MLB Stats CLI App

This is a command-line MLB statistics application written in Python. It is intended to showcase my ability to 
consume and process data from a RESTful API, handle JSON responses, and present live data in a structured CLI interface. 
The program communicates directly with the MLB Stats REST API and was developed and tested using Python and VS Code.
